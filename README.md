# Apple Quantitative Financial Analysis & Algorithmic Trading Research

Institutional-style quantitative finance research project analyzing Apple (AAPL) using forecasting, algorithmic trading, benchmark comparison, and risk-adjusted performance evaluation in R.

---

# Project Overview

This project demonstrates a complete quantitative financial analytics workflow using Apple stock market data retrieved from Yahoo Finance API.

The project includes:
- technical analysis
- ARIMA forecasting
- GARCH volatility modeling
- CAPM analysis
- benchmark comparison against SPY
- algorithmic trading strategy evaluation
- walk-forward validation
- transaction-cost backtesting
- risk-adjusted performance metrics
